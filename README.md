# Entrepreneurs' Mentor Agent
<li>It's a multi-agent system with the agents Mentor, Strategy and Root.</li>
<li>Followed Sequential Agent functionalities.</li>
<li>Mentor agent uses google search tool.</li>
<li>Strategy agent works like strategic brain of the Mentor agent.</li>
<li>Also deployed to Cloud Run service on google cloud.</li>

<h3>Uses :</h3>
It can be used by aspiring entrepreneurs for gaining knowledge regarding the business strategies. It's just a tool which is handy for learning, not a substitute to the human mentor.
